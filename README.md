Módulo de Ranking Académico con Árbol AVL

Descripción del programa:

Este programa implementa un Árbol AVL en C++ para almacenar estudiantes
ordenados por skill_score.

El sistema permite:

- Insertar estudiantes
- Mostrar ranking académico
- Buscar estudiantes por skill_score
- Mostrar estadísticas del árbol AVL

El proyecto fue desarrollado utilizando clases y funciones para mantener
una estructura modular y facilitar su integración posterior dentro del
proyecto final.

Estructura del proyecto:

/semana-avl
    main.cpp

Explicación técnica:

Un Árbol AVL es mejor que un BST normal porque se mantiene balanceado
automáticamente.

Esto evita que el árbol se vuelva muy largo y lento al insertar muchos datos.

Gracias al balanceo, las búsquedas e inserciones son más rápidas y eficientes.

 Nota:

El proyecto fue desarrollado en un compilador en línea, por lo que toda
la implementación se realizó en un solo archivo .cpp.
